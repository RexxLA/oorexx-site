Open Object Rexx Project Status
March - 2005

Web Site

The new ooRexx Web Site has been built, and the ooRexx domains; oorexx.org, oorexx.net, oorexx.com and oorexx.info now point there.

Interpreter Status

After a lot of really good work by the project team we have finally released Version 3.0.0 of Open Object Rexx. You can download the source and binaries for all the supported platforms here. Due to a lack of resources we were not able to complete the AIX port. The Windows binary version includes the PDF documentation but all other platforms will have to download the documentation seperately.

The source for Open Object Rexx is now publicly available and the SourceForge CVS repository is now active.

Documentation

The Version 3.0.0 documentation is available to download at the location listed above.

The source for the Open Object Rexx documentation is now publicly available and the SourceForge CVS repository is now active.

Project Administration

The proposed Project Charter has been accepted by the Development Team and has been submitted to the Rexx Language Association for approval. It will be voted on at the RexxLA Symposium in April, 2005.

Initial roles for the current project team have been defined. Who is doing what can be seen on the ooRexx Contacts page.

Other

Work was completed on the RxFtp class and was included in Version 3.0.0.

As always you can subscribe to the oorexx-announce mailing list to receive development updates and news like this.
