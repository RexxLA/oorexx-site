Open Object Rexx Project Status
July - 2007

Web Site

As of April 27 there had been 45349 visitors to the ooRexx website. This averages to 151 visitors per day.

The ooRexx Build machine website is now up and at least partially operational. It is still not able to produce automated builds yet due to an API problem with VMware and Linux guest operating systems. We are still working through that problem, but the site does now host copies of the manual builds the team is producing.

General

As of Apri 23 Open Object Rexx 3.1.2 is generally available. We have received a large number of downloads for ooRexx and are very pleased with the reception by the community.

A flurry of work has been accomplished towards version 3.2.0. See the details below.

No date has been set for the release of version 3.2.0.

Interpreter Status

Wow! The team has added a lot of new features to ooRexx that will be available in version 3.2.0. There are both new classes and a general reordering of the collection classes. There are also some new builtin functions an new language features. Mark Miesfeld has done some great work on ooDialog and provided some new samples and made it easier to work with.

Documentation

The documentation of all the ooRexx classes in the reference manual has been completely reorganized and updated. Work is continuing on this effort due to the changing organization of the classes.

Project Administration

David Ashley (with approval of the RexxLA board) has taken over the project Management role from Mark Hessling. The Rexx Labguage Association and the ooRexx team members want to thank Mark for his past management of the ooRexx project.

Related News

Work contiues on the ooRexx 3.2.0 release. This will include a number of language enhancements thanks to Rick McGuire.

Work is also proceeding with the future 4.0 release, although at a slow pace. There is still no projected date for this release but progress is being made.

Other

As always you can subscribe to the oorexx-announce mailing list to receive development updates and news like this.
