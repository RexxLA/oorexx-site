Open Object Rexx Project Status

Web Site

We have gotten a lot of positive feedback on the new look and feel of the web site. We will continue to monitor the feedback in this area and make adjustments as necessary.

General

The beta version of ooRexx 4.0 is available. Refer to the announcement page for more details. We hope the final release will be in June but that might change as the beta program progresses.

The team will be actively monitoring bugs opened against the ooRexx 4.0 beta. Hopefully the number will be small as our alpha program was pretty successful in eliminating a number of bugs.

Documentation

The documentation for the 4.0.0 release is about complete and is included in the 4.0 beta. .
