Open Object Rexx Project Status
October - 2006

Web Site

We added a visitor counter to the ooRexx Web Site. This will help us to determine the visibility of Open Object Rexx on the Internet.

General

Open Object Rexx 3.1 is now generally available. We have received a large number of downloads for ooRexx and are very pleased with the reception by the community.

Interpreter Status

A few bugs in the Windows OLE support have been discovered and fixed. We are determining how and when to release an updated version.

The Subversion repository has been set up. It will be used to house the source for the 4.0 version of ooRexx. The CVS repository will continue to house the 3.x versions.

If you want to help just drop a line to the oorexx-devel mailing list.

Documentation

Version 3.1.0 of the documentation is available. This update corresponds to the ooRexx 3.1 release.

Project Administration

The ooRexx Development Team added a new member. Please welcome Mark Miesfeld to the team. Mark has contributed a number of fixes and enhancements to the Windows OLE support and will also be working on the Windows scripting support. Welcome Mark!

Related News

Rony has an official release of his Open Office interface package for ooRexx (BSF4Rexx). This will allow ooRexx to be used as an Open Office macro language. You can get a copy from his BSF4Rexx web site.

Other

As always you can subscribe to the oorexx-announce mailing list to receive development updates and news like this.

Submitted by
W. David Ashley for the Open Object Rexx Project Team
